Task Management Backend API

This project is a backend-focused REST API designed to manage users, projects, and tasks with proper authentication and authorization. It emphasizes clean architecture, security best practices, and scalable API design, reflecting real-world backend development patterns.

Key features include:

JWT-based authentication

Role-based access control (Admin, Member)

Project and task management

Task assignment, priorities, and due dates

Activity logging for auditability

Pagination and filtering for scalable APIs

This repository represents Phase 1 (MVP), with future phases planned for background jobs, rate limiting, and advanced auditing.
